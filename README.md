# Blup-Launcher
> [!IMPORTANT]  
> This app is non functional as it already assumes "C:\\Program Files (x86)\\BlupLauncher\\Shortcuts for BlupLauncher\\1.lnk" is already exists, even when it does not. This app is only public for archival purposes.

**Blup-Launcher** is a simple app made to open games that are part of the Blupi/Eggbert series. (It's only able to open SB and SB2 for now)
### Download
You can download this app on [Sourceforge](https://sourceforge.net/projects/bluplauncher/) or the [releases tab](https://github.com/quythestupid/BlupLauncher/releases/).
